# King of figthers '93

A versus game for two players on a single keyboard. Spaghetti coded for **Stunfest** in 2018, as an easteregg on their website. 

**Design & Code:** Jankenpopp

**Basic Controls**

| Player 1 – Gief | keys    |
|----------|-------------|
| ↑ ← ↓ →  | W A S D  |
| action   | Q       |
| Player 2 – Honda | keys    |
|----------|-------------|
| ↑ ← ↓ →  | arrows   |
| action   | SPACE       |

**Special Moves** (character on left)

| Both players  | move  |
|----------|-------------|
| Dash (Invincible)    | → →  |
| Backdash (Invincible) | ← ←  |
| **Player 1 Gief**  | **move**      |
| Fireball  | ↓ ↘ → Q   |
| Tatsu     | ↓ ↙ ← Q    |
| **Player 2 Honda**       | **move**            |
| Fireball        | ↓ ↘ → SPACE     |
| Honda Press     | ↓ 2sec ↑ SPACE       |
