# Nimoloop 3.0

A Web tracker by Jankenpopp.

### What is it?

Nimoloop is a sample-based music tracker for the web, heavily inspired by Nanoloop and LSDJ softwares, as well as Korg and Teenage Engineering musical instruments, and everything else I like about the various music software packages I've been using for over 25 years.  

It's been designed so that techno music can be written quickly and simply with a mouse and keyboard, with a strong emphasis on live performance. The use of a wheel mouse is strongly recommended.
