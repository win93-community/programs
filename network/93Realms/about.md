# 93 Realms
Multi-User Dungeon for Windows93.net  
[telnet windows93.net:8082](telnet://windows93.net:8082)  


**Game Design / World Building:**  
[Morusque](https://www.nurykabe.com).  


**Main Dev / Editor Tools:**  
[Jankenpopp](http://jankenpopp.com).  


**Game Engine:**  
Based on SimpleMUD by [Long Nguyen](https://github.com/lnguyenfx).    
The original codebase for SimpleMUD was written in C++ by [Ron Penton](https://github.com/RonPenton), the author of MUD Game Programming book.  


**Telnet Client:**  
[Raymond Xie](https://github.com/mudchina/webtelnet).  


**How to play:**  
- First create your character.  
- Then type 'help' for a list of commands
