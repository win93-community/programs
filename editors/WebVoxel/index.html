Controls
  • ESCAPE = Toggle palette mode
  • left click = add a voxel
  • right click = remove a voxel
  • W,A,S,D = move
  • SPACE = jump
  • A,W = float in the air
  • G = land (gravity)
  • R = random color
  • M = Toggle mass
  • L = Reload model
  • alt + left click = copy color
  • ctrl + left click = repaint a voxel
  • "Save" = save a creation
  • drag a file = open a creation
