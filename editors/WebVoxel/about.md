Voxel editor written in JavaScript.  
Coded by [Jankenpopp](http://jankenpopp.com) 


## Controls

- **ESCAPE** = Toggle palette mode
- **Left click** = add a voxel
- **Right click** = remove a voxel
- **W, A, S, D** = move
- **SPACE** = jump
- **A, W** = float in the air
- **G** = land (gravity)
- **R** = random color
- **M** = toggle mass
- **L** = reload model
- **Alt + Left click** = copy color
- **Ctrl + Left click** = repaint a voxel
- **"Save"** = save as .vox
- **Drag a file** = open a .vox
