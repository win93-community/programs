# PowerNoob (Lite)

Draw the web like the noob you are.  

### Draw mode 
- left-click: add element
- right-click: erase element
- Mouse wheel's click+drag: move element

### Text mode 
- left-click: edit element's text
- right-click: edit element's source
- Mouse wheel's click+drag: move element

### Move mode (when you don't have a wheel mouse)
- left-click: move element

### Code mode
- here you can edit the source code of your project

### Preview mode
- test your project here before saving it

### Keyboard Macros
- ctrl+z: undo/redo
- ctrl+t: edit default text
- ctrl+i: edit default image
- ctrl+j: edit default javascript
- ctrl+e: switch modes

### Mouse Macros
The use of a wheel mouse is recommended.
- Mouse wheel zoom-in/zoom-out element
- Mouse wheel's double click : Draw/Text mode switch
